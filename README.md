# @js-ak/db-manager

It is a tool for managing databases.

## Installation

You can install the package via npm:

```
npm install @js-ak/db-manager
```

## Features

* **Typescript**
* **Connect to multiple database types (MySQL, PostgreSQL)**
* **Simple and intuitive interface**
